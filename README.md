## üöÄ Getting Started

This project is powered by **Vite**, providing a fast development environment and optimized production builds.

### Prerequisites

- **Node.js**: Version 18.0 or higher
- **Package Manager**: npm, yarn, or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ikennaphill/VITE-Admin-Dashboard-with-Tailwind-CSS.git
   ```
2. Navigate to the project directory:
   ```bash
   cd VITE-Admin-Dashboard-with-Tailwind-CSS
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## üõ† Available Scripts

In the project directory, you can run:

### `npm run dev`
Runs the app in development mode.  
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.  
The page will hot-reload (HMR) instantly when you make edits to the code.

### `npm run build`
Builds the app for production to the `dist` folder.  
Vite uses **Rollup** under the hood to bundle your assets and optimize them for the best performance. The build is minified, and filenames include hashes for caching.

### `npm run preview`
Locally preview the production build.  
Use this to test if the build works correctly before deploying.

### `npm run lint`
Runs ESLint to find and fix code style issues based on your configuration.

## üåê Deployment

The easiest way to deploy your Vite app is to use the [Vercel](https://vercel.com/) or [Netlify](https://www.netlify.com/) platforms:

1. Push your code to GitHub.
2. Connect your repository to the platform.
3. Ensure the Build Command is `npm run build` and the Output Directory is `dist`.

For manual deployment, simply upload the contents of the `dist` folder to any static web host.
